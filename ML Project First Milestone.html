Hyperparameter Tuning¶

Finding best hyperparameters for logistic regression¶

The parameter solver: {‘newton-cg’, ‘lbfgs’, ‘liblinear’, ‘sag’, ‘saga’}, default=’lbfgs’¶

For small datasets, ‘liblinear’ is a good choice, whereas ‘sag’ and ‘saga’ are faster for large ones so we will consider them.¶

logreg = LogisticRegression()¶

param_grid = [¶

{'C':[0.001,0.003,0.005,0.01,0.03,0.05,0.1,0.3,0.5,1,2,3,3,4,5,10,20],¶

'solver' : ['lbfgs','liblinear'],¶

'max_iter' : [100,1000,2500, 5000]¶

}¶

This is the optimal method to find the best hyperparameters for LR but it requires very high computional power that we don't own so we minimzed the combinations and tested manually each and the default ones gave best results but we tuned the C value to be optitmum¶
